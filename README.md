# EV-DASHBOARD-USING-TABLEAU
# 🚗 Electric Vehicle Data Analysis Dashboard

## 📌 Project Overview
This project is an interactive Tableau dashboard that analyzes Electric Vehicle (EV) adoption across different states, manufacturers, model years, and CAFV eligibility.

The dashboard helps users understand EV market trends through key performance indicators and interactive visualizations.

---

## 📊 Dashboard Features

### KPI Cards
- Total Vehicles
- Average Electric Range
- Total Battery Electric Vehicles (BEV)
- Total Plug-in Hybrid Electric Vehicles (PHEV)

### Visualizations
- 🌎 Total Vehicles by State (Map)
- 📈 Total Vehicles by Model Year (Trend Analysis)
- 🚘 Top 10 Vehicle Makes
- ⚡ Total Vehicles by CAFV Eligibility

## 📈 Key Insights

- Vehicle adoption has increased significantly in recent years.
- A few manufacturers dominate the EV market.
- Battery Electric Vehicles (BEV) have a higher share than Plug-in Hybrid Electric Vehicles (PHEV).
- EV distribution varies considerably across different states.
- CAFV eligibility provides additional insights into vehicle classification.
